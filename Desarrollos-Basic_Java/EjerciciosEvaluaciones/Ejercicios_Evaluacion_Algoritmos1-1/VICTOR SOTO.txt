Desarrollo ejercicio, expresión del problema
Primero que nada es escribió una cadenas de alternativas, para que el usuario buscara la opción de la cual quieres una solución, para nuestra caso son el resolver distintos problemas planteados en el ejercicio:

1.- Para este primer problema donde se debía ingresar solo 3 números y dar promedio se utilizo la función “para” ya que primero debemos ingresar los números y posteriormente cumplida esta condición nos daría un resultado, que para nuestro caso seria una promedio de los números ingresados. Para ayudar a construirlo debimos definir distintas variables ya que algunas empezaban de valor cero. A medida que la función era recorrida, los valores iban siendo guardados en nuestro arreglo, pero además de guardarse se iban sumando y al final se promediaría el total con el numero de datos ingresados.

2.- Para este nuevo problema, debimos definir cual era la variables que queríamos considerar, también debemos considerar que ambos eran textos por lo cual no podíamos considerar algún valor numérico. Se utilizo la función “Si entonces”, ya que la pregunta era sencilla y solo teníamos 2 resultados posibles.

3.- Para este problema también consideramos la opción “Si entonces”, ya que el problema del usuario era definir que iba a utilizar, no importaba el genero de la persona, ya que tanto el baño como la ducha tenían un valor definido sin importan el genero de este. Y ante esto las opciones solo eran 2.

4.- La definición del problema no era el ingreso de los números sino el contar las palabras de cada unos de ellos, una solución practica para este caso fue utilizar la función “según”, ya que de esta manera al ingresar un numero que seria una variable numérica, recorrería la función buscando el igual y dando el resultado ya definido de las palabras para cada numero.
