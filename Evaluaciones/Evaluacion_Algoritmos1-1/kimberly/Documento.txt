En el algoritmo se crea un do-while, una vez termine éste, se termina el Algoritmo. 

En el do se crea un menú (Dado por un switch case) con sus respectivos números y funciones escritas. Para que el menú sea visible se muestran por pantalla con antelación los casos (Nota: Se utiliza el "\n" para saltarse la linea actual a la siguiente).
Do se seguirá usando hasta que se ingrese el número 10, finalizando así el programa



Función 1: Se asignan las variables numéricas x, y, z para recibir los valores por teclado y luego operar con ellos mediante una suma y posteriormente un promedio (Linea 36)

Función 2: Se registran las variables "nombre" y "genero", se designa a la persona el 
baño correspondiente con un "si-entonces"

Función 3: Se designa la variable numérica "Cobro" y se llama a la Función 2. El cobro aumenta dependiendo de las opciones que se dan después de presionar 1 o 2

Función 4: Se asignan los numeros del 1 al 10 mediante un switch case, cada caso tiene su numero en palabras, se designa el número 11 para salir del switch case y cerrar así la función

Función 5: Se designan las variables cont, suma, promedio y numero como numericas. Se crea un do-while para que se repita la acción de recibir número y sumar todos los numeros registrados dentro del do, cada vez que termine un do, el contador aumentará en 1. El bucle do se reiterará hasta que se escriba "0", una vez hecho esto, se efectuará y mostrará el promedio operado con el resultado de suma/cont

Función 6: Se crea un do-while para que se ingrese un nombre, cada vez que no se ingrese "juan" o "Juan" el contador aumentará en 1. Si se ingresa, se mostrará por pantalla el contador como número de intentos

Función 7: Se crea un arreglo con dimensión 10 y los datos ingresados serán caracteres
Se crea un ciclo for, para registrar los nombres en cada índice. Seguido se crea otro for con el propósito de recorrerlo e igualar si el indice actual ya se ha repetido con un índice distinto al actual. Si ya se repitió una vez, entonces mostrará por pantalla "Se repite más de una vez"

Función 8: Se designan las variables N y contador como numéricas, y nombre como caracter. Una vez ingresadas el número total de personas a escribir (N), se asigna N como la dimensión del arreglo "Personas". Se crea un ciclo for que recorra toda la dimensión "Personas" con el fin de ingresar el nombre al índice actual y mostrar por pantalla el contenido del índice con una coma y espacio, a esto se le suma +1 al contador con el fin de ver si el contador es algun divisor de 5. Si el resto entre el contador y 5 da 0, entonces se saltará una linea y seguirá con el siguiente índice

Función 9: Se designan como caracteres las variables "nombre", "apellido" y "email", y como numericos "fono" y "N", este último con el fin de indicar el número total de personas a ingresar. Se inicia un ciclo for para ingresar dato por dato y luego mostrar todo estos datos en una sola linea. Una vez realizado esto, se seguirá con la siguiente persona hasta alcanzar N-1
